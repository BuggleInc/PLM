Bienvenus dans la PLM

L'objectif de ce premier exercice est de faire avancer votre buggle d'un pas. Alternez entre les vues à droite pour plus de détail: Monde montre le monde tel qu'il est pour l'instant, tandis que Objectif montre ce que vous devez obtenir.

Ma buggle ? Qu'est ce que c'est ??

Les buggles sont de petites bêtes qui obéissent aux ordres que vous leur donnez. Pour l'instant, vous devez utiliser le langage [!thelang/] pour cela, mais vous pouvez en changer dans le menu ou en cliquant sur l'icone dans la barre du bas.

Comment faire ?

Écrivez votre programme dans l'onglet "Code". Il devrait ressembler à ceci :

[!java|scala|python]avance()[!java];[/!][/!][!c]avancePas();[/!]
[!java|c]

N'oubliez pas le ; final, car cela indique au compilateur que l'instruction est terminée (oui, les ordinateurs sont si stupides qu'il faut leur préciser des choses aussi simples).

[/!]

Une fois terminé, cliquez sur «Exécuter» pour lancer votre programme.

Si votre code contient des erreurs, c'est tout à fait normal : même les meilleurs programmeurs se creusent la tête sur leur code. L'ordinateur affichera les messages d'erreur sur la console pour vous aider à les résoudre. Les messages affichés peuvent paraître inquiétant au premier abord, mais pas de panique. Le compilateur est seulement pas très doué pour communiquer, il n'est pas fondamentalement méchant. En regardant de plus près, la solution pour corriger les problèmes est inscrite dans ces messages cryptiques. Vous verrez, avec un peu d'entraînement, vous vous y habituerez.
