Ilha dos Buggles

Você acaba de receber as anotações de viagem da sua velha tia que atravessou o país das Ilhas dos Buggles. Ela escreveu a altitude a cada quilômetro durante a viagem, resultando numa sequência de números como segue. 
0 0 1 1 2 2 1 1 0 1 2 2 1 1 0

Aqui está um corte das ilhas por este caminho:

       /***\       /*\         <-- 2 metros acima do nível do mar
   /***********\ /*******\     <-- 1 metro acima do nível do mar
***************************    <-- nível do mar
| | | | | | | | | | | | | |
0 0 1 1 2 2 1 1 0 1 2 1 1 0    <-- altitude em cada ponto

Este exemplo é constituído de quatro níveis de ilhas:

  • 1 1 2 2 1 1 (a ilha da esquerda)
  • 2 2 (o topo da montanha da ilha da esquerda)
  • 1 2 1 1 (a ilha da direita)
  • 2 (a montanha da direita)

Isto também pode ser visualizado desta forma:

0 0(1 1(2 2)1 1)0(1(2)1 1)0

Você poderia escrever um pequeno programa que calculasse a quantidade de níveis numa dada sequência de altitudes? Na verdade, existem uma solução iterativa e uma recursiva para este problema.
