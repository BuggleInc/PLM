Ocorrência de um valor

Neste exercício, você deve calcular a quantidade de ocorrências de um dado valor numa array (ou seja, a quantidade de tempo que este valor aparece na array). Para isto, preencha o método occurrences(), que retorna o número de ocorrências de lookingFor em tab.

Para calcular este valor, simplesmente parcorra a array contando a quantidade de células contendo o valor procurado. Você vai precisar de uma variável extra para isto.
