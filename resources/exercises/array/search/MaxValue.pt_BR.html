Valor maximal

Neste exercício, você deve calcular o valor maximal contido numa array. Para isto, preencha o método maximum(), cujo parâmetro é a array para explorar.

Para calcular este valor, percorra o parâmetro todo. Para cada valor, se ele for maior que o maior valor até o momento, você deve gravar este valor em algum lugar para lembrar dele mais tarde. Você vai precisar então de uma variável extra, que pode ser iniciada para o valor da primeira célula do array.
