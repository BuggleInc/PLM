ìndice do valor máximo

Neste exercício, você deve calcular o índice da célula da tab contendo o maior valor.

Para isto, complete o método indiceDoMáximo(). Seu parâmetro é a array para explorar. Caso a array contenha o valor buscado várias vezes, você deve retornar o índice da primeira ocorrência.

Para resolver este exercício, você deve varrer a array completamente. Para cada valor, se ele for maior que o maior até o momento, este passa a ser o novo maior e você guarda o valor e a posição dele. Você precisa de 2 variáveis extras; o valor inicial do maior pode ser o valor da primeira célula.
