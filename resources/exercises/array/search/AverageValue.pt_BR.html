Valor médio

O objetivo deste exercício é calcular o valor médio dos valores inteiros armazenados numa array.

Você tem que preencher o corpo do método media() que pega como parâmetro a array de inteiros da qual ele calcula e retorna o valor médio. Por favor note que este método deve retornar um inteiro.

para calcular o valor médio de um inteiro, é necessário percorrer toda a array e calcular a soma de todos os valores (logo, você precisa de uma variável para armazenar este valor temporário), e por fim dividir esta soma pelo tamanho da array.

Em Java, você pode obter o tamanho de uma array minhaarray ao consultar seu atributo length (tamanho) (em outras palavras, minhaarray.length). Observe que não existe parêntesis depois do length.

Em python, recuperar o tamanho de uma array minhaarray é tão fácil como chamar len(myarray).
