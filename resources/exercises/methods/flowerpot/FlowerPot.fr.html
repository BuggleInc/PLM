Pot de fleur

Votre buggle a décidé de fleurir un peu le pot de fleur dans lequel elle vit. Vous devez l'aider à reproduire le motif dont elle rêve (vous pouvez le voir dans l'onglet «Objectif»).

For that, you have to write a growFlowers() method that does not take any parameter and does not return any result. You could probably make it without defining this method, but this is part of the assignment.

Also, you should try to decompose this complex drawing in sub-steps so that your code remains short and easy to read. A method to draw one flower of the color passed in parameter sounds like a good start, but it is probably not enough.

For your information, this drawing uses six colors that are listed below. [!java|scala|c]The data type of these constants is Color, that naturally describes a particular color.[/!]

  • [!scala|java|python]Color.[/!]RED pour le rouge
  • [!scala|java|python]Color.[/!]PINK pour le rose
  • [!scala|java|python]Color.[/!]CYAN pour le bleu clair
  • [!scala|java|python]Color.[/!]ORANGE pour le orange
  • [!scala|java|python]Color.[/!]GREEN pour le vert
  • [!scala|java|python]Color.[/!]YELLOW pour le jaune
