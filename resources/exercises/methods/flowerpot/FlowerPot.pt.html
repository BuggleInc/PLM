Vaso de Flores

Seu buggle decidiu plantar umas flores no pote onde vive. Você tem que ajudar ele a reproduzir o desenho dos sonhos dele (verifique a aba "Objetivo" para ver o sonho).

Para isto, você tem que escrever um método growFlowers() que não recebe parâmetros e nem retorna nenhum resultado. Como o padrão é um pouco complexo, você deve tentar decompor este desenho em sub-passos de forma que seu código permaneça curto e fácil de ler. Um método para desenhar uma flor da cor passada como parâmetro parece um bom começo, mas provavelmente não é o suficiente.

For your information, this drawing uses six colors that are listed below. [!java|scala|c]The data type of these constants is Color, that naturally describes a particular color.[/!]

  • [!scala|java|python]Color.[/!]PINK
  • [!scala|java|python]Color.[/!]CYAN
  • [!scala|java|python]Color.[/!]ORANGE
  • [!scala|java|python]Color.[/!]GREEN
  • [!scala|java|python]Color.[/!]YELLOW
