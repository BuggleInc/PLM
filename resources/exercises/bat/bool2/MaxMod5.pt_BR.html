MaxMod5

Sejam dois valores inteiros, retorne o maior dos dois. Entretanto, se os dois valores possuem o mesmo resto quando divididos por 5, retorne o menor dos dois. Entretanto, em todo caso, se os dois valores forem iguais, retorne 0. Nota: o operador % "mod" calcula o resto, i.e. 7 % 5 vale 2.
