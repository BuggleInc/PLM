bilhete verde

Você tem um bilhete de loteria verde, com os inteiros a, b e c inscritos nele. Se os números são todos diferentes uns dos outros, o resultado é 0. Se todos os números são idênticos, o resultado é 20. Se dois dos números são os mesmos, o resultado é 10.
