Baggle Seeking

The buggle world can sometimes contain some baggles, which are little biscuits that buggles can carry from one point to another. For that, they have to use specific methods such as isOverBaggle(), isCarryingBaggle(), pickupBaggle() or dropBaggle(). Click on the button API to check their documentation for more details.

Exercise goal

Let each buggle find its baggle by adapting the code you wrote in previous exercise (copy/paste what you've done before if you want).
