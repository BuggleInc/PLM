Os turmites

Este conjunto de atividades é para você brincar com formigas de Langton, que são máquinas de Turing 2D. Elas constituem problemas bem simples, bons para iniciantes, e abrem as portas para um mundo fantástico.

Este mecanismo foi inventado em 1986 por Chris Langton, e depois generalizado em várias formas (como veremos nos próximos exercícios). Foi provado que Turmites e máquinas de Turing têm o mesmo poder: Uma trajetória de formiga pode ser usada para calcular qualquer circuito booleano, e logo uma formiga é capaz de qualquer cálculo universal. resumindo, qualquer cálculo possível pode ser alcançado usando uma turmite como dispositivo computacional. Ainda outro assunto fascinante...

formigas de Langton multicoloridas foram descobertas em 1995 por Propp et Al. Outro fato divertido é que as formigas cujos nomes é uma lista de pares consecutivos de letras idênticas (LL e RR) produzem padrões simétricos. Este fato foi até mesmo provado formalmente.

Verifique a página da wikipédia correspondente, na qual este exercício é inspirado, para maiores detalhes.
