Baseball

Ceci est actuellement en cours de développement, et sera bientôt parfaitement intégré à JLM. Merci de votre patience.

A l'instar du monde précédent, ce monde implémente un monde de baseball. Mais maintenant, vous avez plus de deux joueurs sur chaque base. Comme auparavant, votre mission est d'amener chaque joueur à sa base.

Cet exercice essaye de vous montrer une chose : vous avez à faire un changement que vous n'aviez pas anticipé, et il faut le faire sans casser votre code et sans le modifier trop en profondeur, puisque votre algorithme est toujours correct.

L'algorithme précédent était seulement capable de gérer deux joueurs par base, vous avez à le modifier afin qu'il puisse gérer n joueurs par base, avec n supérieur ou égal à deux.

Vous pouvez vous demander pourquoi vous n'avez pas eu à écrire directement un algorithme sans ce défaut. C'est juste que parfois, c'est beaucoup trop difficile. Parfois, il suffit de prendre un cas particulier et de généraliser la solution. C'est bien souvent plus simple de généraliser que de trouver un algorithme général à partir de rien. Souvenez-vous qu'améliorer est plus facile que créer.

Notes:

  • Vous pouvez avoir accès au nombre de positions disponibles sur chaque base en utilisant la méthode getLocationsAmount().
  • La couleur d'une base est donnée par son index.
  • Le trou a une couleur de -1.
  • La base qui a un seul joueur sur le terrain a getAmountOfBases()-1 comme index

Sept méthodes vous sont fournies : 

void move(int baseSrc, int playerSrc)
Déplace le joueur de la position playerLocation de la base baseSrc au trou. 

int getPlayerColor(int baseIndex, int playerLocation)
Renvoie la couleur du joueur à la position playerLocation ( 0 ou 1 ) de la base baseIndex. La couleur -1 correspond au trou. 

int getHoleBase()
Renvoie l'index de la base où se trouve le trou 

int getHolePositionInBase()
Renvoie la position de joueur vide dans la base où se situe le trou 

boolean isBaseSorted( int baseIndex )
Renvoie VRAI si tous les joueurs sur la base baseSrc sont sur la bonne base. 

int getAmountOfBases()
Renvoie le nombre de bases dans le terrain 

int getLocationsAmount()
Renvoie le nombre de positions de joueur disponible sur chaque base du terrain
