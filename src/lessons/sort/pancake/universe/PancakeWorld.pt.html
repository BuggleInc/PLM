PancakeWorld

Este universo é bastante simples, com apenas cinco funções fornecidas.

[!java|c]int [/!]getStackSize()  [!scala]:Int[/!]
Retorna o tamanho da pilha, ou seja, a quantidade de panquecas nela. 
[!java|c]int [/!]getRaioPanqueca([!java|c]int [/!]rank[!scala]:Int[/!])  [!scala]:Int[/!]
Retorna o raio da panqueca passado como argumento, com o rank da panqueca mais acima a ser 0. 
[!java]boolean [/!][!c]int [/!]panquecaDeCabeçaPraBaixo([!java|c]int [/!]rank[!scala]:Int[/!])   [!scala]:Boolean[/!]
Retorna se a panqueca passada como argumento está de cabeça para baixo, ou seja, se o lado queimado está para cima. Como sempre, a panqueca mais acima é a de rank 0. 
[!java|c]void [/!]flip([!java|c]int [/!]amount[!scala]:Int[/!])
Vira as primeiras amount panquecas que compõe a pilha, do topo dela. 
[!java]boolean [/!][!c]int [/!]isSorted()   [!scala]:Boolean[/!]
Retorna true se a pilha de panquecas estiver corretamente ordenada. 
[!java]boolean [/!][!c]int [/!]isSelected()   [!scala]:Boolean[/!]
Retorna true se o mundo atual está selecionado na interface gráfica.
