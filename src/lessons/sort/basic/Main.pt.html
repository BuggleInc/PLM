Algoritmos de Ordenação

Esta lição permite a experimentação com alguns algoritmos de ordenação clássicos (e algumas variações menos comuns deles). E temos um duplo objetivo: pode primeiro compreender melhor a ideia destes algoritmos a escrever eles por si só. Mas mesmo se não codificar os algoritmos, pode usar o modo de demonstração para organizar "corridas" entre os algoritmos para experimentar na prática o que significam diferentes complexidades assintóticas.

Mais exercícios estão previstos para o futuro, sobre os algoritmos de ordenação recursivos (como o QuickSort e o MergeSort) ou a usar outros micromundos para aplicar estes algoritmos a outros contextos.
