All Different

Dada uma [!scala]List[Int][/!][!java|python]recursive list[/!], retorne true se quaisquer dois elementos da lista são diferentes (ie, se nenhum elemento aparece mais de uma vez na lista).

A solução mais simples é em O(n²), o que significa que executa em tempo quadrático. Provavelmente terá que definir uma função adicional para isto. Outra solução muito mais complexa executa em O(n log(n)).
