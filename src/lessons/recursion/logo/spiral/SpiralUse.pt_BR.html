Desenhando espirais

Este exercício não é realmente um exercício pois você tem que simplesmente clicar no botão Rodar para ter uma recompensa: todo o código necessário já está pronto. Na verdade, você pode explorar as possibilidades do código que você acabou de escrever mudando os parâmetros da chamada da função spiral. Você pode mover para o próximo exercício se estiver com pressa, e voltar depois para experimentar este.

Aqui estão algumas figuras que você pode produzir com sua função Spiral, dado que você passe os parâmetros certos. Você pode trocar para o modo criativo (do menu Exercício) para desabilitar a comparação de mundos quando estiver explorando com estas espirais.

Se você descobrir uma figura legal, não esqueça de enviar seus parâmetros (menu Ajuda/"Dizer o que achou") para que possamos adicionar a esta lista!
