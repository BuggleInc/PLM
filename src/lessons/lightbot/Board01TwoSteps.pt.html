Bem-vindo

Bem-vindo ao mundo lightbot. Este é apenas um quebra-cabeça para programadores ao invés de uma lição real (embora algumas pessoas usem isto para ensinar programação). O robô não é programado em Java, mas sim graficamente. Pode ver as ordens existentes na documentação a usar o menu Sobre este mundo.<

O objetivo de cada tabuleiro é simplesmente ligar todas as luzes do tabuleiro a usar o seu pequeno robô.
