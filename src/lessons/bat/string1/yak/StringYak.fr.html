String Yak

Supposer la chaîne "yak" comme malchanceuse. Soit une chaîne de caractères, renvoyez une version où tous les "yak" sont supprimées, mais le "a" peut être n'importe quel caractère. La "yak" chaîne ne sera pas pétinée.
