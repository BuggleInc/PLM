
# Somme d'ados #
Étant donné deux nombres entiers (a et b), retournez leur somme. Cependant,
les valeurs ados (dans l'intervale [13;19]) sont particulièrement
chanceuse. Donc, si l'un des nombres est ado, retournez simplement 19.
