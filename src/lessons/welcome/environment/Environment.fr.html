Welcome to the PLM

The goal of this first exercise is to instruct your buggle to move forward by one step. Switch between the World view and the Objective one for more details.

My buggle? What is this??

Buggles are little animals obeying any order you may give them. Currently, you must use the [!thelang/] programming language for that, but you can change the language from the menu, or by clicking the icon in the status bar.

How to proceed?

Write the code to move your buggle by one step forward in the Code pane (using the interactive controls is not enough). Your code should read as follows:

[!java|scala|python]avance()[!java];[/!][/!][!c]avancePas();[/!]
[!java|c]

N'oubliez pas le ; final, car cela indique au compilateur que l'instruction est terminée (oui, les ordinateurs sont si stupides qu'il faut leur préciser des choses aussi simples).

[/!]

Once done, click on Run to execute your code.

If your code contains errors, that's perfectly OK: Even talented programmers sometimes struggle on their code. The computer will display error messages in the console to help you fixing them. The displayed messages may sound scary at first glance, but don't panic. The compiler is only somehow limited in its communication abilities, but it's not mean. If you look closer, the solution to solve your issue is written somewhere in those cryptic messages. You will see, with a bit of habit, we get used to it.
