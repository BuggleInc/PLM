Bem-vindo ao PLM

O objectivo deste primeiro exercício é instruir o seu buggle para avançar um passo. Alterne entre as vistas à direita para mais informações. O Mundo retrata o mundo como é agora, enquanto o Objetivo retrata o que deve obter.

O que é o meu buggle?

Os buggles são pequenos animais que obedecem todas as suas ordens. Atualmente, deve usar a linguagem de programação [!thelang/] para isso, mas pode mudar a linguagem a partir do menu, ou a clicar no ícone na barra de estado.

Como proceder?

Escreva o seu programa no painel "Código". Deve ser lido da seguinte forma:

[!java|scala|python]forward()[!java];[/!][/!][!c]stepForward();[/!]
[!java|c]

Não se esqueça do ; final que diz ao compilador que a instrução está encerrada (sim, o computador é tão bobo que não consegue adivinhar coisas óbvias como isto).

[/!]

Quando feito, clique em Executar para executar o seu código.

Se o seu código contém erros, tudo bem: até os programadores talentosos lutam com o seu código. O computador exibirá mensagens de erro na consola para o ajudar a corrigi-los. Estas mensagens podem parecer assustadoras à primeira vista, mas não entre em pânico. O compilador é apenas de alguma forma limitado nas suas habilidades de comunicação, mas não é mau. Se olhar mais de perto, a solução para resolver o seu problema está escrita em algum lugar nessas mensagens crípticas. Verá, com um pouco de hábito, que nos acostumamos a isso.

Mas isso é [!thelang]!

Atualmente, o PLM está configurado para aceitar programas [!thelang], mas também pode aceitar programas [!java]Python ou Scala[/!][!scala]Java ou Python[/!][!python]Java ou Scala[/!]. Clique no ícone da linguagem no canto inferior direito da janela, ou vá para o item de menu Language/Computer para mudá-lo.
