Pot de fleur

Votre buggle a décidé de fleurir un peu le pot de fleur dans lequel elle vit. Vous devez l'aider à reproduire le motif dont elle rêve (vous pouvez le voir dans l'onglet «Objectif»).

Pour cela, vous devez écrire une méthode nommée growFlowers() («fait pousser les fleurs» en anglais). Cette méthode ne prend aucun paramètre et ne renvoie aucun résultat. Vous pourriez probablement réaliser ce motif sans cette méthode, mais l'écrire fait partie de l'exercice.

De plus, vous devriez décomposer ce dessin complexe en sous-motifs pour que votre code reste simple et agréable à lire. Une méthode dessinant une fleur est un bon début, même si ce n'est probablement pas assez.

Pour information, ce dessin utilise les cinq couleurs ci-dessous. [!java|scala|c]Le type de données de ces constantes est Color, qui décrit naturellement une couleur donnée[/!]

  • [!scala|java|python]Color.[/!]RED pour le rouge
  • [!scala|java|python]Color.[/!]PINK pour le rose
  • [!scala|java|python]Color.[/!]CYAN pour le bleu clair
  • [!scala|java|python]Color.[/!]ORANGE pour le orange
  • [!scala|java|python]Color.[/!]GREEN pour le vert
  • [!scala|java|python]Color.[/!]YELLOW pour le jaune
