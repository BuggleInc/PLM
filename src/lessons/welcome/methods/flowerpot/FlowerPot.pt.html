Vaso de Flores

O seu buggle decidiu plantar umas flores no pote onde vive. Tem que ajudá-lo a reproduzir o desenho dos sonhos dele (verifique a guia "Objetivo" para ver o sonho).

Para isso, tem que escrever um método growFlowers() que não toma nenhum parâmetro e não retorna nenhum resultado. Provavelmente poderia fazê-lo sem definir este método, mas isto é parte da atribuição.

Além disso, deve tentar decompor este desenho complexo em sub-passos para que o seu código permaneça curto e fácil de ler. Um método para desenhar uma flor da cor passada no parâmetro parece um bom começo, mas provavelmente não é suficiente.

Para sua informação, este desenho usa cinco cores que estão listadas abaixo. [!java|scala|c]o tipo de dados destas constantes é Cor, que naturalmente descreve uma cor particular.[/!]

  • [!scala|java|python]Cor.[/!]ROSA
