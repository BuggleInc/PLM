ìndice do valor máximo

Neste exercício, deve calcular o índice da célula da tab a conter o maior valor.

Para isto, complete o método indexOfMaximum(). O parâmetro dele é a array para explorar. Caso a array contenha o valor procurado várias vezes, deve retornar o índice da primeira ocorrência.

Para resolver este exercício, deve varrer a array completamente. Para cada valor, se for maior que o maior até o momento, este passa a ser o novo maior e guarda o valor e a posição dele. Precisa de 2 variáveis adicionais; o valor inicial do maior pode ser o valor da primeira célula.
