Ocorrência de um valor

Neste exercício, deve calcular a quantidade de ocorrências de um dado valor numa array (ou seja, a quantidade de tempo que este valor aparece na array). Para isto, preencha o método occurrences(), que retorna a quantidade de ocorrências de lookingFor em tab.

Para calcular este valor, simplesmente percorra a array a contar a quantidade de células a conter o valor procurado. Vai precisar de uma variável adicional para isso.
